# BustOS
This is our attempt at making an operating system from scratch.
# Planned Features in Order of Importance:
1. Welcome message
2. Read keystrokes
3. Able to type text on screen
4. Recive input and return output
5. 32 bit real mode
6. More than 1 sector
7. Able to code in c
8. Shell
9. Cowsay
